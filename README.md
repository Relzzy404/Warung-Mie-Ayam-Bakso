# 🍜 Warung Mie Ayam & Bakso 

Aplikasi Flutter sederhana untuk mengelola **draft produk** berbasis API.

---

## 🚀 Fitur Utama

- 🔐 Login menggunakan API
- 📦 Menampilkan daftar produk (GET)
- 🗑️ Menghapus produk (DELETE - soft delete)

---

## 🧠 Konsep yang Digunakan

- REST API (GET, POST, DELETE)
- Token Authentication (Bearer Token)
- Stateful Widget
- HTTP Request (package http)
- Soft Delete (data tidak dihapus permanen)
